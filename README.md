# Contador de Casamento — Front-end Simples

Abra o arquivo `index.html` em um navegador para ver o site (nenhum servidor é necessário).

O que tem aqui:
- `index.html` — estrutura do site com o contador, dois carrosséis e duas declarações de amor.
- `styles.css` — estilos usando tons de laranja.
- `script.js` — lógica do contador e carrosséis.

Como alterar a data do casamento:
1. Edite o atributo `data-wedding-date` na tag `<body>` em `index.html`.
2. Use o formato ISO: `YYYY-MM-DDTHH:MM:SS` (ex.: `2026-06-12T15:00:00`).

Imagens: atualmente são usados placeholders públicos (picsum.photos). Substitua os `src` das imagens em `index.html` por URLs das suas fotos, ou adicione imagens locais na pasta e aponte para elas.
